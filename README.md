# CCNA Home Lab (Physical Network Lab)

## Overview
This repository documents my hands-on CCNA home lab built with real Cisco hardware.  
I am actively configuring and testing networking concepts using CLI tools like PuTTY while studying for the CCNA 200-301 exam.

This lab focuses on applying networking theory in a real environment, including switching, routing, and firewall configuration.

---

## Lab Environment

- **Management Device:** Dell Inspiron 3721  
- **Switch:** Cisco Catalyst 2960X-Series  
- **Router:** Cisco 1900 Series  
- **Firewall:** Cisco ASA 5506-X  
- **Rack:** 20U Network Rack  
- **Patch Panel Installed**

---

- Build hands-on experience with Cisco networking equipment
